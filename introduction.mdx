## Tech

ShipShack makes use of the following components to help you get started:

1. [**Next.js**](https://nextjs.org) - React framework for building websites
2. [**Tailwind CSS**](https://tailwindcss.com) - Utility-first CSS framework
3. [**Auth.js**](https://authjs.dev/) - Authentication
4. [**PostgreSQL**](https://www.postgresql.org) - Database for storing your data
5. [**Drizzle**](https://orm.drizzle.team/) - TypeScript ORM for interacting with your database
6. [**tRPC**](https://trpc.io) - TypeScript RPC framework

#### Services

You will require an account with the following services. Each of these offer a generous free tier for you to get started:

1. [**Stripe**](https://stripe.com) - Payment processing
2. [**Vercel**](https://vercel.com) - Hosting
3. [**PostHog**](https://posthog.com) - Analytics
4. [**Crisp**](https://crisp.chat) - Live customer chat and support
5. [**SendGrid**](https://sendgrid.com) - Email delivery for authentication and marketing
6. [**Sentry**](https://sentry.io/) - Error tracking

<Info>**Why do I need to sign up for these services?** While ShipShack is an excellent starter point for a SaaS application, it tries not to reinvent the wheel. These services provide an excellent foundation for your app and offer excellent free tiers.</Info>
